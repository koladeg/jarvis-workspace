## HOW TO USE THIS

**Every Sunday (19:00 WAT):**

1. **Check WhatsApp group activity** (last 7 days)
   - Count messages per agent
   - Note dormant/active agents
   
2. **Check Abe Emu status** (recruitment database)
   - Who confirmed training?
   - Who sent bank details?
   - Who went silent?

3. **Fill in the template** above with current data

4. **Post to Telegram** group (`Jarvis-AdugboInsure`)

## DATA SOURCES (For Dashboard Accuracy)

- **Apete agent activity:** WhatsApp group messages (manual count or export)
- **Abe Emu status:** Master agent list (banking details, form responses)
- **Training dates:** Your calendar / OYSHIA confirmation
- **Content posted:** Weekly script posting tracker
